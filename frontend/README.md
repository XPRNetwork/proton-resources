# XPR Network Resources

## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Testnet

```sh
yarn dev
```

### Compile and Hot-Reload for Mainnet

```sh
yarn dev:prod
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
